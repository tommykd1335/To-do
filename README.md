# TechTask Pro - Gestión Financiera de Trabajos

Una aplicación web moderna y intuitiva para gestionar trabajos, ingresos y gastos de manera profesional.

## Características

- **Registro de Trabajos**: Agrega trabajos con descripciones, fechas y categorías.
- **Gestión de Ingresos y Gastos**: Para cada trabajo, registra múltiples ingresos y gastos.
- **Métricas en Tiempo Real**: Visualiza ingresos, gastos y ganancias diarias, semanales y mensuales.
- **Historial Completo**: Accede a un historial detallado de todas las transacciones.
- **Exportación a PDF**: Exporta el historial a un archivo PDF para registros permanentes.
- **Búsqueda y Filtrado**: Busca trabajos por descripción, fecha o categoría.
- **Edición y Eliminación**: Modifica o elimina trabajos existentes.
- **Interfaz Moderna**: Diseño responsive con gradientes, animaciones y colores intuitivos.

## Tecnologías Utilizadas

- **HTML5**: Estructura de la aplicación.
- **CSS3**: Estilos modernos con Bootstrap 5.3.3 y animaciones personalizadas.
- **JavaScript ES6+**: Lógica de la aplicación con manipulación del DOM y localStorage.
- **Bootstrap 5**: Framework CSS para componentes responsive.
- **FontAwesome**: Iconos para una mejor experiencia visual.
- **html2pdf.js**: Para exportar el historial a PDF.

## Instalación y Uso

1. Clona o descarga los archivos del proyecto.
2. Abre `index.html` en un navegador web moderno.
3. La aplicación funciona completamente offline, guardando datos en el localStorage del navegador.

## Funcionalidades Detalladas

### Agregar un Trabajo
- Completa la descripción, fecha y categoría.
- Agrega ingresos y gastos dinámicamente con descripciones y montos.
- Guarda el trabajo para calcular métricas automáticamente.

### Ver Métricas
- Tarjetas con ingresos/gastos diarios, semanales y mensuales.
- Ganancias netas calculadas en tiempo real.

### Historial
- Modal con lista detallada de todos los trabajos.
- Opción para exportar a PDF.

### Editar Trabajo
- Modal para modificar trabajos existentes.
- Agrega o elimina ingresos/gastos adicionales.

### Búsqueda
- Campo de búsqueda para filtrar trabajos.
- Botón para limpiar la búsqueda.

## Diseño y UX

- **Profesional**: Gradientes suaves, sombras y transiciones.
- **Intuitivo**: Formularios claros, colores codificados (verde para ingresos, rojo para gastos).
- **Responsive**: Funciona en desktop, tablet y móvil.
- **Accesible**: Etiquetas, placeholders y navegación clara.
